# Important Branches
Please take note of branches of the project. The important branches are following:
* `main` - contains latest development version
* `release` - contains latest release version

Please do pull reguest on `main` branch.

# Release Schedule
GM-MPM follows semester release schedule (every six-month), according to Indonesian academic calendar, starting in 2025:
* `year.01` released at the end of June
* `year.02` released at the end of December

If there are new features added after the scheduled release date, said new features will be released at the next release date.

For the time being, after all the core functionalities has been made, it will be released immediately after testing and debugging.
